# D-VMD-and-D-MVMD
De-mixing (multivariate) variational mode decomposition (D-VMD, D-MVMD)

Author: Jiawei Jian 

This function is modified from Variational Mode Decomposition proposed by Konstantin Dragomiretskiy and Dominique Zosso.
